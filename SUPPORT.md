# Support

## Documentation

The primary resource for all questions is the official documentation:

**[docs.stratarouter.com](https://docs.stratarouter.com)**

It covers installation, quick start, API reference, integrations, architecture,
deployment, and troubleshooting guides.

---

## Community Support (Free)

### GitHub Issues
Use [GitHub Issues](https://github.com/ai-deeptech/stratarouter/issues) for:
- Bug reports
- Feature requests
- Documentation issues

Please search existing issues before opening a new one.

### GitHub Discussions
Use [GitHub Discussions](https://github.com/ai-deeptech/stratarouter/discussions) for:
- General questions and usage help
- Design and architecture discussions
- Sharing what you've built with StrataRouter
- Announcements and roadmap feedback

### Bug Reports
When filing a bug, please include:
- StrataRouter version: `python -c "import stratarouter; print(stratarouter.__version__)"`
- Python version: `python --version`
- Operating system
- Minimal reproduction script
- Full traceback

---

## Email Support

For questions not suited to public forums:

**[support@stratarouter.com](mailto:support@stratarouter.com)**

Response time: within 2 business days.

---

## Security Vulnerabilities

**Do not** open a public GitHub issue for security vulnerabilities.

Report them privately to **[support@stratarouter.com](mailto:support@stratarouter.com)**
with subject `[SECURITY] Brief description`.

See [SECURITY.md](SECURITY.md) for the full responsible-disclosure policy.
